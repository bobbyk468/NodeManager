Figure 2 — Topological Reasoning Mapping (TRM) — Formal Definitions

Formal Model · Definitions 1–5
Definition 1
Step Mapping σ
A function mapping each extracted concept ci to its nearest KG node nj by lexical + semantic distance.
σ : C_extracted → V_KG ∪ {∅}
Definition 2
Node Adjacency
Two mapped nodes are adjacent if there exists a directed edge between them in the KG (node-only criterion, v11 locked).
adj(n_i, n_j) ⟺ (n_i, n_j) ∈ E_KG
Definition 3
Structural Leap
A gap between consecutive mapped nodes that are non-adjacent in the KG — indicating a conceptual jump in the student's answer.
leap(i) = 1 iff ¬adj(σ(c_i), σ(c_{i+1}))
Definition 4
Leap Count L
Total structural leaps across the full concept sequence. Higher L indicates fragmented reasoning with missing intermediate concepts.
L = Σ leap(i), i ∈ [1, |C|−1]
Definition 5
Grounding Density ρ
Fraction of LRM reasoning steps that cite at least one KG node. Used as a model-quality proxy in stability analysis.
ρ = |{steps: kg_nodes ≠ ∅}| / |steps|
Chain Coverage
chain_pct Score
Primary C5 sub-score. Percentage of expected KG concepts that were matched by the student's extracted concept set.
chain_pct = |σ(C) ∩ V_expected| / |V_expected|
Figure 2. Formal TRM definitions. The adjacency criterion (Def. 2) uses node-only edges — intermediate edge labels are not required to match student text. This is a deliberate design choice that makes the model robust to paraphrasing while still penalising conceptual leaps (Def. 3–4).

Interpretation: The monotonic decline in KG benefit from Mohler → DigiKlausur → Kaggle ASAG is consistent with a domain-boundary hypothesis: ConceptGrade's advantage diminishes as the target domain vocabulary exceeds the coverage of auto-generated KGs. Kaggle ASAG is not a failure — it defines the operating boundary of the system.
Figure 5. Domain complexity moderates KG benefit.
